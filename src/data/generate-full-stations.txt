|Name|Image|Type|Kbps|Votes|Clicks|Location|Language|Link|Website|

{
  id: "sequential_number",

  type: "MP3/AAC/etc",
  kbps: "bitrate",
  votes: numeric_value,
  clicks: numeric_value,
  location: "India or specific city",
  language: "hindi/tamil/etc or empty string",
  link: "streaming url",
  website: "website url"
}

Default values if missing:
